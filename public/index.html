Projects

PaintBar

A modern, web-based paint application that provides an intuitive drawing experience with a clean, minimalist interface. PaintBar combines the simplicity of traditional paint programs with modern web technologies.

Hashgraph

A revolutionary distributed ledger technology implementation.

Pandas Who Code is a main contributor for the CI/CD pipelines and automations within Hashgraph.

Hiero-Ledger

Hiero, a Linux Foundation Decentralized Trust project, is an open-source, vendor-neutral distributed ledger technology. Hiero is used to build the Hedera public ledger.

Roger Barker is a maintainer for the Hiero-Ledger project.

TacSimVR

TacSimVR was a startup project that intended to provide immersive tactical simulation and training for law enforcement and military professionals.

The project died early on, but we learned a lot.

GH Audit Setup

GitHub Audit Setup is a script that automates the setup of GitHub repositories for security and CI/CD audits.

Setup Git Semver

Github Marketplace

Git Semver Setup Action is a PandasWhoCode maintained github action that enables the setup-git-semver cli tool to be used within a github workflow.

Read All Custom Properties

Read All Custom Properties is a PandasWhoCode maintained github action that reads all custom properties from the repositories in a github organization and outputs them to a yaml file in the repository.

Update Custom Properties

Github Marketplace

Custom Properties Update Action is a PandasWhoCode maintained github action that updates the custom properties in all repositories based on the values defined in the input file.

By default, it will use the file repo-properties.yaml in the root of the repository.
